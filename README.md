# Asteroids
Juego inspirado en el clásico Asteroids usando HTML, CSS y JavaScript (Canvas).

## Controles
- Flechas: mover nave
- Espacio: disparar
- Enter: iniciar / reiniciar

## Arquitectura
MVC:
- Model: estado del juego
- View: renderizado
- Controller: lógica y control
